# Performance-and-Risk-Analysis-of-UBS-and-the-MSCI-World-ETF-with-GARCH-Based-Portfolio-Allocation
An in depth analysis of UBS vs MSCI World CHF hedged ETF with stylized facts of returns, QQ plot normality checks, multi method VaR forecasting and violation backtesting, time varying volatility modelling, and VaR based dynamic portfolio allocation in R and LaTeX.
